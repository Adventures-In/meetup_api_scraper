Photo Album List

GET /:urlname/photo_albums
  • json
Host: api.meetup.com
  • basic
api version 3

Gets a list a group photo albums in ascending order based on the time they were created

Try it in the console

Request Parameters

A valid path parameter value for :urlname is required

This endpoint uses HTTP Link header based pagination.

fields

A comma-delimited list of optional response fields. Currently supported values are "self" and "short_link"

offset

Incrementing number used for pagination offsets

page

Number of albums to return per page of results
