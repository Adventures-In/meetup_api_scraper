Get Pro Network RSVPs

GET /pro/:urlname/rsvps
  • json
Host: api.meetup.com
  • basic
api version 3

Get the list of RSVP responses for the events across the entire Pro network

Try it in the console

Request Parameters

offset

Integer: offset of page, starting from 0

page

Integer: size of page. Minimum of 1 and maximum of 200

Response

if successful, 200 with the JSON array, ordered by mtime, recent first

event_id

Integer: Unique identifier for the event

group_id

Integer: Unique identifier for the group for the event

member_id

Integer: Unique identifier for the member who RSVPed

mtime

Long Integer: Unix timestamp of the last time RSVP response was modified

rsvp_id

Integer: Unique identifier for the rsvp response

status

String: Status of the RSVP response. One of yes, no, and waitlisted
