Event comment delete

DELETE /:urlname/events/:event_id/comments/:comment_id
  • json
Host: api.meetup.com
  • group_content_edit
api version 3

Deletes comments posted in events

Request Parameters

A path parameter for :urlname, :event_id, and :comment_id is required.

The :comment_id path parameter should be formatted as comment:{comment_id} for top level comments or reply:{reply_id} for comment replies

To obtain a list of comments and their replies for an event, see the event comments endpoint

Only organizers or the member that posted the comment may delete the comment

In addition, authorized oauth applications should request the group_content_edit permission scope.

Response

A successful request will result in an HTTP 204 NoContent response
