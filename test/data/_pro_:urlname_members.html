Search Pro Members

GET /pro/:urlname/members
  • json
Host: api.meetup.com
  • basic
api version 3

Name, location, and time based search for the members of the meetups belonging to Pro organization.

Try it in the console

Request Parameters

All parameters are optional.

active_within_days

Positive Integer: The range of date from the past until today, for the recent activity

chapters

List of Integers: The chapters which the member belongs to

country

String: Country

desc

Boolean: whether to sort ascending or decending

email_received

Integers: The id of a previous emails that the member received

events_attended_max

Positive Integer: Maximum number of attended events

events_attended_min

Positive Integer: Minimum number of attended events

is_organizer

Boolean: To limit to only organizers or non-organizers

join_time_max

Milliseconds since epoch: The latest time limit for member join

join_time_min

Milliseconds since epoch: The oldest time limit for member join

lat

Float: Latitude

location

String: Raw location

lon

Float: Longitude

member_name

String: Name of the member

offset

Positive Integer: the page offset

order

String: attribute to sort on

page

Positive Integer: the size of page window

query

String: raw query string to search from member name or city

radius

String: global, smart, or search radius in Float

zip

String: Zip code

Ordering

events_attended

By the number of events attended

is_organizer

By whether the member is an organizer or not

join_time

By the time when the member joined Meetup

last_access_time

By the time of most recent visit

member_name

By the member name

Response

chapters

Pro organization groups that the member belongs to

id

Id of the group

name

Name of the group

urlname

Urlname used to identify the group on meetup.com

city

City of the member

country

Country of the member

email

Email address of the member if the member opted to share it with the organization

events_attended

The number of attended events

is_organizer

Organizer status of the member

join_time

The time when the member joined Meetup

last_access_time

The time when the last activity occured

lat

Latitude

lon

Longitude

member_id

Id of the member

member_name

Name of the member

photo_thumb_url

Url of the photo thumbnail of the member

state

State of the member, if in US or Canada
