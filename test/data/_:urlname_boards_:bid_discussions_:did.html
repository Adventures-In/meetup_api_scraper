Discussion Posts

GET /:urlname/boards/:bid/discussions/:did
  • json
Host: api.meetup.com
  • basic
api version 3

Listing Group discussion posts

Try it in the console

Request Parameters

The :urlname path element may be any valid group urlname or domain name. The :bid path element maybe any valid board ID for this group. The :did may be any valid discussion ID for this board

Response

body

content of the post

created

Time post was created in milliseconds since the epoch

discussion

The discussion this was posted in

id

Numeric discussion ID

id

Numeric post ID

in_reply_to

ID of the post this was in reply to

member

The member that started this discussion

city

City, if provided, for the member

country

Country, if provided, for the member

id

Member ID

name

Member name

photo

Photo object for active member

thumb

Thumbnail photo url

state

State if in the US

subject

subject of the post

updated

Time post was updated in milliseconds since the epoch
