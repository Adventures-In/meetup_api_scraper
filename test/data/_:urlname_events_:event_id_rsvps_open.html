Open Rsvps

POST /:urlname/events/:event_id/rsvps/open
  • json
Host: api.meetup.com
  • event_management
api version 3

Opens rsvps for an event

Request Parameters

Valid path parameters for :urlname and :event_id are required.

Only event hosts and members of the lead team may open rsvps.

OAuth authenticated applications should request the event_management OAuth scope.

Response

A successful request will return a 204 response.
