Get Group Member Profile

GET /:urlname/members/:member_id
  • json
Host: api.meetup.com
  • basic
api version 3

Gets Group Profiles. For Member Profiles, see this endpoint

Try it in the console

Request Parameters

A valid path parameter for :urlname and :member_id is required. A value of "self" may be used in place of a numeric identifier to represent the authenticated Member's id

fields

A comma-delimited string of optional response field names. This may include birthday, gender, groups, privacy, self and topics

Response

Returns a single Member Profile object with embedded group Profile information. If the group's content is restricted to groups members and the authenticated member is not a member of the group, a 403 Forbidden error response will be returned.

Edit Group Member Profile

PATCH /:urlname/members/:member_id
  • json
Host: api.meetup.com
  • profile_edit
api version 3

Edits Group Profiles. To fetch Group Member Profiles, see this endpoint

Request Parameters

A valid path parameter for :urlname and :member_id is required. A value of "self" may be used in place of a numeric identifier to represent the authenticated Member's id.

OAuth authenticated applications should request the profile_edit permission scope

Only organizers of the Meetup group may edit the member's title and role within the group.

All other fields are restricted to the owner of the group profile.

Some Meetup group organizers may define a set of profile questions they'd like their members to answer. You can obtain that question list using the Get Group by sending a "fields" request parameter containing "join_info"

add_role

Allows those with permission to assign one of the following roles: "assistant_organizer", "coorganizer", or "event_organizer"

answer_{qid}

Answers to questions from group's API join_info question fields.

The name of these parameters must contain a suffix of _ + the numeric question id from the Meetup group's join info question list

Answers may not be longer than 1000 characters in length

fields

A comma-delimited string of optional response field names. This may include birthday, gender, groups, privacy, self and topics

intro

Provides a Member an opportunity to tell the group about themselves, in at most 255 characters

photo_id

Numeric id of the photo to use for this profile. Send a value of 0 to unset the current photo

remove_role

Allows those with permission to remove one of the following roles: "assistant_organizer", "coorganizer", or "event_organizer"

title

An organizer-defined title, in at most 255 characters

Response

Returns a single Member Profile object with embedded group profile information. If a request is made that violates one of the field restrictions mentioned above, a 403 Forbidden error response will be returned.

Delete Group Member Profile (Leave Group)

DELETE /:urlname/members/:member_id
  • json
Host: api.meetup.com
  • profile_edit
api version 3

Deletes a member's group profiles.

Request Parameters

A valid path parameter for :urlname and :member_id is required. A value of "self" may be used in place of a numeric identifier to represent the authenticated Member's id.

OAuth authenticated applications should request the profile_edit permission scope

Only the authenticated member is permitted to leave groups they do not organize. For groups they do organize, they must first step down on the groups homepage on www.meetup.com

exit_comment

Optional message to the organizer when leaving

Response

A successful request will result in a 204 NoContent response.

Attempts to delete profiles that do not belong to the authenticated member will result in a 403 Forbidden response

Attempts to delete profiles in groups the authenticated member is an organizer of will result in a 400 BadRequest response

Errors

organizer_error

Returned when an organizer attempts to delete their profile before stepping down
