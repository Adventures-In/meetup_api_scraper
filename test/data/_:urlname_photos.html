Album Photos

GET /:urlname/photos
  • json
Host: api.meetup.com
  • basic
api version 3

Lists of all photos uploaded for the group

Try it in the console

Request Parameters

A valid path parameter for :urlname is required

This endpoint uses HTTP Link header based pagination

desc

Controls directional order or listing. Default false

fields

A comma-delimited list of optional response fields. Currently supported values are "comment_count", "self" and "short_link"

page

Number of items to return per-page of results. Defaults to 200
