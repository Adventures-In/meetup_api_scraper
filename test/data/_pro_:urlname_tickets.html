Search Issued Pro Tickets

GET /pro/:urlname/tickets
  • json
Host: api.meetup.com
  • basic
api version 3

Redeemed chapter, redeemed member, status, and timestamp for tickets belonging to Pro organization.

Try it in the console

Request Parameters

All parameters are optional.

chapter_name

String: partial or full name of the redeemed chapter

chapters

Comma delimited positive integers: ids of redeemed chapters

country

String: two-letter country code of the redeemed chapters

status

Comma delimited strings: ticket status, among the following: inactive, available, claimed, redeemed

ticket_key

String: full matching string of an issued ticket key

Ordering

chapter_name

By the name of the chapter that redeemed the ticket

created

By the creation timestamp of ticket

status

By the redeemed status of the ticket

updated

By the update timestamp of ticket

Response

chapter

Pro organization chapter that redeemed the ticket, if the ticket is redeemed

country

Country of the Chapter

id

Id of the chapter

name

Name of the chapter

urlname

Urlname used to identify the chapter on meetup.com

created

Ticket creation time

member

Member who claimed the ticket, if the ticket is claimed

id

Id of the member

name

Name of the member

status

The status of ticket, one of the following: inactive, available, claimed, redeemed

ticket_key

Ticket key

updated

Last update time of ticket
