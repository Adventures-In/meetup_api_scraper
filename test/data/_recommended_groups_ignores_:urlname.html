Recommended Groups Ignore

POST /recommended/groups/ignores/:urlname
  • json
Host: api.meetup.com
  • basic
api version 3

Provides a form of feedback by requesting to remove a group from future recommendations

Request Parameters

Only the path parameter :urlname is required. You are limited to 100 of these requests in a 24 hour period of time.

Response

A successful request will result in a 201 Created or 204 No Content response. If you go over the rate limit documented for this service you will receive a 429 Too Many Requests responses. Otherwise, a 400 Bad Request response is returned
