Photo Comment Delete

DELETE /:urlname/events/:event_id/photos/:photo_id/comments/:comment_id
  • json
Host: api.meetup.com
  • group_content_edit
api version 3

Deletes photo comments

Request Parameters

A URI containing a valid group urlname, event_id, photo_id, and comment_id is required.

The authenticated member must be the original poster of the comment or an admin of the group

In addition, authorized oauth applications must request the group_content_edit permission scope.

Response

A successful HTTP DELETE request will yield a 204 No Content response. Otherwise a 400 Bad Request response can be expected
