Get Pro RSVP registration responses

GET /pro/:urlname/rsvps/survey
  • json
Host: api.meetup.com
  • basic
api version 3

Get the list of RSVP registration responses for the events across the entire Pro network

Try it in the console

Request Parameters

cursor

String: identifier for the next result page

events

String: Comma separated list of event id. Max: 99 items

size

Integer: amount of results for one request. Max: 1000 items

Response

cursor

Identifier for next result page

results

List of registration responses

answers

List of answers to event survey questions asked when the member RSVP'd in the order asked, only available to organizers and assistant organizers

question

Translated question title

response

Answer

chapterName

Group name

email

Member email

eventTitle

Event title

updatedAt

Timestamp when registration was updated
