Member Calendar

GET /self/calendar
  • json
Host: api.meetup.com
  • basic
api version 3

Get a listing of all upcoming Meetup events for the authenticated member

Try it in the console

Request Parameters

All parameters are optional.

This endpoint uses HTTP Link header based pagination

A comma-delimited list of optional fields names which may be appended to the response
