Photo Comment

POST /:urlname/events/:event_id/photos/:photo_id/comments
  • json
Host: api.meetup.com
  • group_content_edit
api version 3

Creates a new photo comment

Request Parameters

Valid path parameters for :urlname, :event_id, and :photo_id are required.

You must be a member of the group to post comments on the photo.

In addition, authorized oauth applications must request the group_content_edit permission scope.

comment

The text of the comment

Response

comment

The comment the member left for the photo

created

Time the comment was posted in milliseconds since the epoch

id

Comment id

member

The member that made the comment.

In cases where member has since left the group, this member may have an id of 0

bio

Intro of member

event_context

Member's context within the event. Only returned in the context of an event

host

Boolean indicator for whether this member is a host for the event

id

Numeric member ID

name

Name of member

photo

Member photo, if available. Members who registered via Facebook may not have highres photos

base_url

A base url that can be use to construct a photo url from its components

highres_link

Link for full sized photo

id

Numeric photo ID

photo_link

Link for standard sized photo

thumb_link

Link for thumbnail sized photo

type

Type of photo. One of "event" or "member"

role

The leadership role of this member within the group, if any. Value may be one of assistant_organizer, coorganizer, event_organizer, organizer

title

Organizer defined title of member. May be absent if not defined

self

Context for the authenticated member

actions

A list of actions the authenticated member may perform on the comment.

Value may be one or more of the following:

delete delete this comment

Photo Comments

GET /:urlname/events/:event_id/photos/:photo_id/comments
  • json
Host: api.meetup.com
  • basic
api version 3

Lists photo comments associated with a photo

Try it in the console

Request Parameters

Valid path parameters for :urlname, :event_id, and :photo_id are required

For groups that have limited public visibilty on their content, you must be a member of the group to see these
