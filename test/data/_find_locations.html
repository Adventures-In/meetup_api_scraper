Find locations

GET /find/locations
  • json
Host: api.meetup.com
  • basic
api version 3

Provides a query interface for finding known locations

Try it in the console

Request Parameters

This endpoint uses HTTP Link header based pagination.

lat

Search for locations based on location latitude. Must be provided with "lon"

lon

Search for locations based on location longitude. Must be provided with "lat"

offset

The current offset in the paginated set, represented as an incrementing value

page

The desired number of locations to return in a single set of results. Defaults to 200

query

Search for locations based on city name or zip code

Response

The order the results are returned in is based on the request parameters provided. By default, the nearest results based on "lat" and "lon" provided will be returned first, otherwise location with the highest member density will be returned first.

city

Name of city

country

Two character country code

lat

Approximate location latitude

localized_country_name

Localized name of country based on request's language information

lon

Approximate location longitude

name_string

The full name of the location as preformatted string

state

Enclosing location state, if the location's country declares one

zip

The location's zip code. For locations in countries without zip codes, a placeholder will be returned
